# Pierogi Quest

A simple 3D platformer created in Godot. Explore the town, help the citizens and find all the collectibles.

## Controls
|Input|Action|
|-----|------|
|WASD|Move|
|Ctrl|Run/Walk|
|Space (hold)|Jump|
|Mouse|Look around|
|E|Interact|
|Esc|Pause|
